# Contributing to Horizon Stays

Thank you for helping improve the Hotel Booking System.

## Steps

1. Fork the repository.
2. Create a branch: `git checkout -b feature/short-description`.
3. Make one focused change.
4. Build and test the affected application.
5. Commit with a clear message.
6. Push the branch and open a pull request.

## Commit Messages

Use messages such as:

- `feat: add room availability filter`
- `fix: refresh bookings after cancellation`
- `docs: improve setup instructions`
- `style: improve room card layout`

## Pull Requests

Explain what changed, why it was needed, and how it was tested. Include screenshots for interface changes. Never commit passwords, API keys, connection strings, build output, or dependency folders.
